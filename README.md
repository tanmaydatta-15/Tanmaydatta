## ✨ The Vision: Offline-First AI
Most AI requires a constant internet connection. My apps are built to be **Resilient and Independent**. 
* **User Interface:** Familiar "Gemini/WhatsApp" style for ease of use.
* **Edge Intelligence:** Powered by Hugging Face models that run locally.
* **Knowledge Base:** Includes a local, offline version of Wikipedia for signal-free learning.

## 🛠️ Tech Stack
* **Framework:** React + TypeScript (Vite)
* **Intelligence:** Google AI Studio (Prototyping) + Hugging Face (Offline Inference)
* **Infrastructure:** Progressive Web App (PWA) for mobile installation.
* **Environment:** Developed in a 900+ Mbps "Mango Lab" cooled by natural tree canopies.
